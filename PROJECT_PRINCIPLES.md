# PROJECT_PRINCIPLES

## 1. Paper First / 纸质优先
系统必须始终知道档案位于哪个柜、哪一层、哪个盒、盒内第几份。

## 2. Archive Before Document / 档案优先于文件
文件只是档案的一部分。档案还包含审批、柜位、借阅、状态、生命周期、附件、权限和日志。

## 3. Digital Must Match Physical / 数字必须匹配现实
系统中的柜位图必须真实反映现实柜体。

## 4. Everything is Metadata / 一切皆元数据
字段、页面、菜单、权限、标签、柜体、数据集、公式、流程、AI知识库都应尽量配置化。

## 5. Preview First / 预览优先
附件的第一动作是在线预览，不是下载。

## 6. AI is Assistant / AI 只是助手
AI 可以建议、解释和生成草稿，但不得直接删除数据、修改权限、销毁档案或批量迁移。

## 7. Audit Everything / 一切关键操作必须留痕
新增、修改、删除、下载、借阅、归还、审批、迁移、AI建议采纳都必须写入不可删除日志。
