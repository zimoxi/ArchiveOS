# RUNBOOK

## Hermes 每次开发流程

1. 进入 ArchiveOS 本地目录。
2. 执行：

```bash
git status
git pull origin main
```

3. 打开：

```text
START_DEVELOPMENT.md
```

4. 按 START_DEVELOPMENT 指定文件阅读，不扫描整个仓库。
5. 执行当前 TASK。
6. 运行必要检查。
7. 提交：

```bash
git add .
git commit -m "<task commit message>"
git push origin main
```

8. 输出报告。
